## Features (Free Edition)

- Folder selection for image sequences
- Rename by pattern (prefix only)
- Keep original numbering and file extensions
- Change order (renumber sequences safely)
- Copy x2 images
- Delete files by even / odd numbers
- Supports: `.png`, `.jpg`, `.jpeg`, `.exr`
